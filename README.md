# AI-Powered Resume Screening and Job Matching System

## Overview
An AI-powered application that analyzes resumes, extracts technical skills, evaluates candidate profiles, and matches resumes with job descriptions.

## Features
- PDF Resume Parsing
- Skill Extraction
- Resume Scoring
- Role-Based Skill Gap Analysis
- Job Description Matching
- Skill Analytics Dashboard
- Downloadable Analysis Reports

## Technologies Used
- Python
- Streamlit
- Scikit-Learn
- PDFPlumber
- Pandas
- Matplotlib
